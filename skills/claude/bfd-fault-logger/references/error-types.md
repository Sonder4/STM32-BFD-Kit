# 错误类型定义

本文档定义了STM32H7系列微控制器中可能出现的硬件错误类型及其详细说明。

## 1. CPU核心错误

### 1.1 HardFault

**描述**: 最严重的系统错误，通常表示不可恢复的故障。

**触发条件**:
- 非对齐内存访问
- 执行未定义指令
- 从无效地址取指
- 访问受保护内存区域
- 其他可配置错误的升级

**相关寄存器**:
| 寄存器 | 地址 | 描述 |
|--------|------|------|
| HFSR | 0xE000ED2C | HardFault状态寄存器 |
| CFSR | 0xE000ED28 | 可配置故障状态寄存器 |

**HFSR 位定义**:
| 位 | 名称 | 描述 |
|----|------|------|
| 31 | DEBUGEVT | 调试事件触发 |
| 30 | FORCED | 强制HardFault（由可配置错误升级） |
| 1 | VECTTBL | 向量表读取错误 |

### 1.2 BusFault

**描述**: 总线访问错误，发生在指令预取或数据访问期间。

**触发条件**:
- 预取指令错误
- 精确数据总线错误
- 不精确数据总线错误
- 堆栈操作错误

**相关寄存器**:
| 寄存器 | 地址 | 描述 |
|--------|------|------|
| BFSR | CFSR[15:8] | BusFault状态寄存器 |
| BFAR | 0xE000ED38 | BusFault地址寄存器 |

**BFSR 位定义**:
| 位 | 名称 | 描述 |
|----|------|------|
| 7 | BFARVALID | BFAR有效标志 |
| 5 | LSPERR | 浮点惰性状态保存错误 |
| 4 | STKERR | 堆栈错误（入栈） |
| 3 | UNSTKERR | 堆栈错误（出栈） |
| 2 | IMPRECISERR | 不精确数据总线错误 |
| 1 | PRECISERR | 精确数据总线错误 |
| 0 | IBUSERR | 指令总线错误 |

### 1.3 UsageFault

**描述**: 指令执行相关错误。

**触发条件**:
- 执行未定义指令
- 无效状态（Thumb位错误）
- 无效PC加载
- 无协处理器
- 非对齐访问
- 除零错误

**相关寄存器**:
| 寄存器 | 地址 | 描述 |
|--------|------|------|
| UFSR | CFSR[31:16] | UsageFault状态寄存器 |

**UFSR 位定义**:
| 位 | 名称 | 描述 |
|----|------|------|
| 9 | DIVBYZERO | 除零错误 |
| 8 | UNALIGNED | 非对齐访问 |
| 3 | NOCP | 无协处理器 |
| 2 | INVPC | 无效PC |
| 1 | INVSTATE | 无效状态 |
| 0 | UNDEFINSTR | 未定义指令 |

### 1.4 MemManageFault

**描述**: 内存保护单元(MPU)违规错误。

**触发条件**:
- 指令访问违规
- 数据访问违规
- 堆栈操作违规

**相关寄存器**:
| 寄存器 | 地址 | 描述 |
|--------|------|------|
| MMFSR | CFSR[7:0] | MemManage状态寄存器 |
| MMFAR | 0xE000ED34 | MemManage地址寄存器 |

**MMFSR 位定义**:
| 位 | 名称 | 描述 |
|----|------|------|
| 7 | MMARVALID | MMFAR有效标志 |
| 5 | MLSPERR | 浮点惰性状态保存错误 |
| 4 | MSTKERR | 堆栈错误（入栈） |
| 3 | MUNSTKERR | 堆栈错误（出栈） |
| 1 | DACCVIOL | 数据访问违规 |
| 0 | IACCVIOL | 指令访问违规 |

## 2. 外设错误

### 2.1 DMA错误

**描述**: DMA传输过程中发生的错误。

**触发条件**:
- 传输错误(TEIF)
- 直接模式错误(DMEIF)
- FIFO错误

**错误标志**:
| 标志 | 描述 |
|------|------|
| TEIFx | 传输错误中断标志 |
| DMEIFx | 直接模式错误中断标志 |
| FEIFx | FIFO错误中断标志 |

### 2.2 UART错误

**描述**: UART通信过程中发生的错误。

**错误标志**:
| 标志 | 名称 | 描述 |
|------|------|------|
| ORE | 溢出错误 | 接收缓冲区溢出 |
| FE | 帧错误 | 停止位检测失败 |
| NE | 噪声错误 | 检测到噪声 |
| PE | 奇偶校验错误 | 奇偶校验失败 |

### 2.3 SPI错误

**描述**: SPI通信过程中发生的错误。

**错误标志**:
| 标志 | 名称 | 描述 |
|------|------|------|
| MODF | 模式错误 | NSS引脚配置错误 |
| OVR | 溢出错误 | 接收缓冲区溢出 |
| CRCERR | CRC错误 | CRC校验失败 |
| UDR | 下溢错误 | 发送缓冲区下溢 |

### 2.4 I2C错误

**描述**: I2C通信过程中发生的错误。

**错误标志**:
| 标志 | 名称 | 描述 |
|------|------|------|
| BERR | 总线错误 | 起始/停止条件检测错误 |
| ARLO | 仲裁丢失 | 多主机仲裁失败 |
| OVR | 溢出错误 | 接收缓冲区溢出 |
| TIMEOUT | 超时错误 | 时钟超时 |

### 2.5 ADC错误

**描述**: ADC转换过程中发生的错误。

**错误标志**:
| 标志 | 名称 | 描述 |
|------|------|------|
| OVR | 溢出错误 | 数据寄存器覆盖 |
| AWD | 模拟看门狗 | 超出阈值范围 |

## 3. 系统错误

### 3.1 看门狗超时

**描述**: 看门狗定时器超时触发的系统复位。

**类型**:
- 独立看门狗(IWDG)超时
- 窗口看门狗(WWDG)超时

**诊断信息**:
| 标志 | 描述 |
|------|------|
| IWDGRSTF | 独立看门狗复位标志 |
| WWDGRSTF | 窗口看门狗复位标志 |

### 3.2 堆栈溢出

**描述**: 堆栈指针超出分配区域。

**检测方法**:
- MSP/PSP边界检查
- 堆栈保护区域检测
- MPU区域违规

### 3.3 时钟错误

**描述**: 时钟系统相关错误。

**错误类型**:
| 类型 | 描述 |
|------|------|
| PLL锁定失败 | PLL无法锁定 |
| HSE故障 | 外部晶振故障 |
| CSS检测 | 时钟安全系统触发 |
| LSE故障 | 低速外部晶振故障 |

### 3.4 电源错误

**描述**: 电源管理相关错误。

**错误类型**:
| 类型 | 描述 |
|------|------|
| 欠压检测(BOR) | 电压低于阈值 |
| 可编程电压检测(PVD) | 可编程电压阈值触发 |
| 稳压器故障 | 内部稳压器错误 |

## 4. 错误严重程度分类

| 级别 | 描述 | 示例 |
|------|------|------|
| Critical | 系统崩溃级错误，需要立即处理 | HardFault, MemManageFault |
| High | 功能失效错误，影响系统功能 | BusFault, DMA错误 |
| Medium | 性能降级错误，部分功能受影响 | 外设通信错误 |
| Low | 信息性警告，不影响系统运行 | 非关键外设警告 |
| Info | 调试信息，仅供参考 | 状态变化日志 |

## 5. 错误源分类

| 源 | 描述 | 典型错误 |
|----|------|----------|
| CPU | 处理器核心错误 | HardFault, UsageFault |
| Memory | 内存系统错误 | MemManageFault, 堆栈溢出 |
| Bus | 总线系统错误 | BusFault |
| DMA | DMA控制器错误 | 传输错误, FIFO错误 |
| Peripheral | 外设错误 | UART/SPI/I2C错误 |
| Clock | 时钟系统错误 | PLL故障, HSE故障 |
| Power | 电源系统错误 | 欠压, 过流 |
| System | 系统级错误 | 看门狗超时 |
